# PortfolioHub - Digital Portfolio Builder

A modern, responsive website for a digital portfolio builder platform. This is a landing page that showcases the features, templates, and pricing of a portfolio building service.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Beautiful gradient backgrounds and smooth animations
- **Easy Navigation**: Sticky navigation bar with smooth scrolling
- **Feature Showcase**: 6 key features displayed with icons and descriptions
- **Template Gallery**: 4 different portfolio templates with preview cards
- **Pricing Plans**: Three pricing tiers (Free, Pro, Business) with feature comparison
- **Contact Form**: Get in touch section for inquiries
- **Interactive Elements**: Click handlers for buttons and forms
- **Analytics Tracking**: Basic event tracking functionality
